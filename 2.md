# CÀI ESINT VÀ PRETTIER

    - Eslint: linter (bộ kiểm tra lỗi) chính V8
    - Prettier: code formatter chính

    - @typescript-eslint/eslint-plugin: Eslint Plugin cung cấp các rule cho typescript
    - @typescript-eslint/parser: parse cho phép eslint kiểm tra lỗi typescript

    - eslint-config-prettier: Bộ config Eslint vô hiệu hóa các rule của Eslint mà xung đột với Prettier
    - eslint-plugin-import: Để eslint hiểu các cú pháp import... trong source code
    - eslint-plugin-jsx-a11y: Kiểm tra các vấn đề liên quan đến accessbility( thân thiện website)

    - eslint-plugin-react: Các rule Eslint cho react
    - eslint-plugin-prettier: Dùng thêm 1 số rule Prettier cho Eslint
    - eslint-plugin-react-hooks: Bắt lỗi react-hooks
