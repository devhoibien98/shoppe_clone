# CÀI SOURCEMAP ĐỂ BIẾT CÁC CSS Ở FILE NÀO

    - Chúng ta vào file vite thêm css là object có thuộc tính là devSourcemap bật true lên là được

# SỬA LỖI SETUP IMPORT

    - Sửa trong file ts.config là baseURL chỉnh lại là thành dấu . hoặc dùng trong path như này "@/*": ["./src/*"]

# CẤU HÌNH ROUTER

    - Cài react-router-dom
    -  Chúng ta sẽ tạo thêm 2 folder là Layouts và Pages, và trong Layouts thì sẽ chứa các Layouts lớn chứa nhiều Page bên trong, còn Page thì sẽ chứa Page trong mỗi Layout
    - Thêm là mỗi folder sẽ tạo thêm 1 file index sau đó trong mỗi file index chúng ta sẽ export ra các file có trong folder đó để import gọn hơn
    - Tạo folder con bằng câu lệnh mkdir folder_mom/folder_con
          file thì bằng câu lệnh touch folder_mom/folder_con/...

# TẠO CUSTOM HOOK ROUTER

    - Dùng useRouters([{}]) Nó là 1 mảng chứa các object và mỗi object thông thường nó sẽ có path và element
    - Biến nhận kết quả từ useRoutes và trả về 1 React Element chứa component nên dùng như biến
    - useRoutes là lấy URL hiện tại

    - Nếu là để layout con nằm trong cha thì dùng children còn liên quan đến đường dẫn cha rồi mới tới đường dẫn con thì dùng outlet
    - Outlet liên quan đến URL, còn Children liên quan đến UI

# INTERFACE

    - Prop truyền vào Component đều phải có interface
        - Children có kiểu là React.ReactNode
    - Miễn không phải là cái gì liên quan đến UI thì đều phải bỏ trong cặp dấu {}
