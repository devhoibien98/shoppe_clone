    - Dùng VITE: yarn create vite
    - Change port: sửa trong file vite.config.ts
        - Thêm 1 object server và có key là port

    - Khi dùng VITE thì nó sẽ yêu cầu Node sẽ từ 14.18+, 16+
        - Kiểm tra máy có bao nhiêu node version: nvm list
        - Kiểm tra máy đang dùng node nào: nvm current
        - Kiểm tra máy bằng node: node -v

    - key target trong tsconfig thì nó sẽ build ra phiên bản năm nào
